PROGRAMMING LANGUAGE BASICS

Before diving into DSA, make sure you are comfortable with any one of these preferred languages C++, Java, Python

TIME COMPLEXITY AND SPACE COMPLEXITY

Learn to analyze complexity of your codes using Asymptotic notations

ARRAYS

Start with the basic array insertions, access, searching and deletion and level up till Sliding Window Technique, Two Pointer technique and Kadane's algorithm

STRINGS

Start with the basic string reversals, palindromes, anagram detection and learn till Robin-Karp Algorithm and KMP algorithm

LINKEDLISTS

Start learning types of Linkedlists and how they work and to perform operations such as Traversal,Insertion and deletion. Then learn some patterned questions like Cycle detection, Reversing a linkedlist, merge sorted linkedlists

STACKS & QUEUES

Start with basic operations such as Push, Pop, peek, enque, deque, front. Then learn its applications like Balanced paranthesis, Next greater element, Min stack

TREES

Learn about various types of trees like BST, Balanced trees, Red-Black tree
